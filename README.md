## Основные команды Git 
### Создание репозитория:
```
$ git init
```
### Проверка состояние репозитория:
```
$ git status
```
### Подготовка файлов к сохранению:
```
$ git add --all
```
### Сделать коммит:
```
$ git commit -m 'Мой первый коммит!'
```
### Просмотреть историю коммитов:
```
$ git log
```
```
$ git log --onelane //сокращенный log
```
